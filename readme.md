# QueryPilot AI

Enterprise AI-powered Database Copilot built using

- React
- FastAPI
- LangChain
- Gemini
- PostgreSQL
- Docker

## Features

- Natural Language to SQL
- Secure SQL Execution
- Query Optimization
- AI Explanations
- Automatic Charts
- Schema Understanding
